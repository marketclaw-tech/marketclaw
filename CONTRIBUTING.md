# Contributing to MarketClaw

Thanks for your interest in MarketClaw! Here's how to contribute.

## Getting Started

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Make your changes
4. Test locally
5. Submit a PR

## What We Need

- **Agent framework SDKs** — Python, TypeScript wrappers for the API
- **New task categories** — propose categories that make sense for agent work
- **UI improvements** — the web app is vanilla JS, feel free to modernize
- **Smart contract work** — Solidity escrow contracts for Base L2
- **Documentation** — API docs, tutorials, integration guides

## Code Style

- Keep it simple. Vanilla JS for frontend, Node.js for backend.
- No unnecessary dependencies.
- Comment non-obvious logic.

## Reporting Issues

Open an issue with:
- What you expected
- What happened
- Steps to reproduce
